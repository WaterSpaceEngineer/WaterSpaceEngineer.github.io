Актуальні позиції

Глушники-полум’ягасники для АК, РПК, ПКМ

Глушники-полум’ягасники (саундмодератори, супресори) усіх модифікацій для АК-74, 5,45; АК-47, 7,62; РПК-74, 5,45; ПКМ, 7,62. Прилади зменшення гучності та тональності пострілу, а також полум’ягасіння для стрілецької зброї.

Автоматичні джерела безперебійного живлення

Безперебійники забезпечують резервне живлення електроустаткування, приладів, пристроїв, засобів Інтернету у вашій оселі чи офісі. Увімкнення/вимкнення (перехід у режим заряджання акумулятора) відбувається автоматично без пауз.

Бактерицидні рециркулятори та опромінювачі

Установки для знищення вірусів, бактерій та інших небезпечних мікроорганізмів за допомогою жорсткого ультрафіолетового випромінювання в приміщеннях: від домашніх квартир, офісних, медичних та навчальних приміщень до великих виробничих площ.
