Глушители-пламегасители для АК, РПК, ПКМ

Походный фильтр для ВСУ и туристических отрядов

Компактный мобильный фильтр обеспечивает комплексную очистку воды из любых водоемов, включая особо загрязненные. Не требует подключения к источникам электроэнергии (подача воды осуществляется встроенным ручным насосом).
