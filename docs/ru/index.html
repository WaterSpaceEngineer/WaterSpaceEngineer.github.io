Глушители-пламегасители для АК, РПК, ПКМ

Глушители-пламегасители (саундмодераторы, супрессоры) всех модификаций для АК-74, 5,45; АК-47, 7,62; РПК-74, 5,45; ПКМ, 7,62. Приборы уменьшения громкости и тональности выстрела, а также пламегашения для стрелкового оружия.
